# ms-arquitetura
 Arquitetura de Microserviços Java Utilizando as tecnologias:
 
 - Spring Boot
 - Spring Batch
 - Spring Cloud
 - Eureka Server (Discovery Server)
 - Hystrix
 - Resilience4J
 - Proxy Zuul
 - Spring Cloud Gateway (Implementação para substituir Zuul)
 - Feing
 - Ribbon
 - Sql ( Oracle e PostgreSQL)
